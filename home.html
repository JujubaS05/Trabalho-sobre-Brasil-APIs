Bem vindos ao nosso site Brasil APIs!!

O objetivo central da Brasil API é superar limitações de acesso programático a informações públicas, como os CEPs (Códigos de Endereçamento Postal), que não podem ser acessados diretamente por um navegador devido à falta de suporte para CORS (Cross-Origin Resource Sharing) na API dos Correios. Dessa forma, mesmo que a fonte dos dados seja lenta, a API fornece acesso rápido e moderno a informações como CEP, DDD, dados bancários, CNPJ, dados do IBGE, feriados nacionais, tabela FIPE, ISBN, registros de domínio .br e taxas.

Para interagir com o nosso site, selecione umas das ações que estão logo abaixo do título e você será direcionado a página que deseja!!

Cada página possui a sua função:

↪ ISBN: É o Número Padrão Internacional de Livro onde você irá procura o livro desejado através do seu código de barra.

↪ CNPJ: É o Cadastro Nacional da Pessoa Jurídica onde você irá procurar dados de empresas pelo seu CNPJ.

↪ IBGE: É o Instituto Brasileiro de Geografia e Estatística onde você irá procurar por todos os Estados ou apenas por um Estado específico.
